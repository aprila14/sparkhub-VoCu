# Python BLE client

### Tested environments
* Ubuntu 22.04
* bluez 5.64-0ubuntu1
* Python 3.10.12

## Start
```bash
pipenv run python3 main.py
```
To Stop Ctrl+C

## Installation

### Prerequisites on Linux
```
pip3 install pipenv==2023.8.28
```
### Pipenv and Pipfile
```bash
pipenv install
```

## Used external Python programming libraries
